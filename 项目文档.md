## 🎨 设计风格

### 视觉风格

| 特性 | 描述 |
|------|------|

2. **极客按钮**
   - 渐变背景（青→蓝）
   - 发光阴影效果
   - 悬停放大动画

3. **聚光灯卡片**
   - 鼠标追踪光晕

4. **漫画风彩蛋**
   - 头像悬停 1 秒触发
   - 漫画气泡对话框
   - 硬阴影 + 粗边框

## 📦 部署指南

### Vercel 部署

1. **推送代码到 GitHub**
   ```bash
   git add .
   git commit -m "feat: 更新首页"
   git push origin main
   ```

2. **在 Vercel 导入项目**
   - 访问 https://vercel.com
   - Import Git Repository
   - 选择 `jhqm/my-website`

3. **自动部署**
   - Vercel 会自动检测 Next.js 项目
   - 每次 push 到 main 分支自动部署

### 自定义域名（可选）

1. **购买域名**（预算 100 元/年，.com 后缀）
2. **在 Vercel 添加域名**
3. **配置 DNS 解析**
